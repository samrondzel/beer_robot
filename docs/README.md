# BeerBot GitHub Pages site

Copy these files into your repository `/docs` folder and deploy with GitHub Pages.

## Structure

```text
docs/
├── index.html
├── journey.html
├── notes.html
├── style.css
└── assets/
```

## After copying

```bash
git add docs
git commit -m "Add BeerBot landing pages"
git push
```

Then GitHub Pages will redeploy automatically.
